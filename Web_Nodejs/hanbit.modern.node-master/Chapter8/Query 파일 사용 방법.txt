Query.txt 파일을 복사하신 후에
MySQL Command Monitor에 명령어를 붙여넣으면
자동으로 데이터가 생성됩니다.

윈도우의 경우는 명령 프롬프트의 테두리에 마우스 커서를 놓고
마우스 오른쪽 버튼을 눌러 [편집]-[붙여넣기]를 누르시면 됩니다.

우분투와 맥의 경우는 그냥 붙여넣으면 됩니다.
